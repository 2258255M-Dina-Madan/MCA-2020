Week 5 - Standards in Curation

Task 1 - Schema of Metadata Elements for Describing Music

  • File Description - describing the computer MEI file:
    • Title Statement - title of file + sub-elements
      • Title of music piece
      • Composer- who created it
      • Responsibility Statement - who encoded the transcribed piece
    • Edition Statement - source of transcription + editorial activity
      • Edition
      • date
      • Revised by
    • Extent - describing the physical size of the file
    • Publication Statement
      • Publisher
      • Publisher location including 'geoname' tag

  • Encoding Description - methods in the ecoding and editing process:
    • Application software that has been used
    • Project information
